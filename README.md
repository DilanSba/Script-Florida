# Windmar Solar - Speech Console

Herramienta estratégica para asesores de **Windmar Solar Florida**. Permite gestionar y aplicar guiones de venta (Speeches) de forma dinámica, optimizando la comunicación en el call center.

## Características

- **Modo Asesor**: Interfaz optimizada para llamadas en tiempo real con sustitución de variables dinámicas (Nombre, Ciudad, Dirección).
- **Modo Líder Comercial**: Consola de administración para crear, editar y eliminar estrategias comerciales.
- **Manejo de Objeciones**: Panel rápido de respuestas para rebatir objeciones comunes.
- **Paleta de Colores Corporativa**: Diseño basado en la identidad visual de Windmar Home Solar.
- **Persistencia Local**: Los guiones se guardan en el navegador para uso inmediato.

## Tecnologías

- **React 19**
- **Vite**
- **Tailwind CSS 4**
- **Lucide React** (Iconografía)
- **Framer Motion** (Animaciones)

## Instalación y Ejecución

1. Clona el repositorio:
   ```bash
   git clone <url-del-repositorio>
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

4. Construye para producción:
   ```bash
   npm run build
   ```
